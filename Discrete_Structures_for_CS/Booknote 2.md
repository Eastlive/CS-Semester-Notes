# 2.1 Set

- **Cartesian product**
  $$A × B = \{(a, b) | a ∈ A ∧ b ∈ B \}$$

- **Power set**
  $$P(A) = \{X | X ⊆ A \}$$

# 2.2 Set Operation

- **union**
  $$A ∪ B = \{x | x ∈ A ∨ x ∈ B \}$$

- **intersection**
  $$A ∩ B = \{x | x ∈ A ∧ x ∈ B \}$$

- **disjoint**
  $$A ∩ B = \emptyset$$

- **difference**
  $$A - B = \{x | x ∈ A ∧ x ∉ B \}$$

- **complement**
  $$\overline{A} = \{x ∈ U | x ∉ A \}$$
